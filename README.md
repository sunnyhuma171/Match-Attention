## Introduction
本仓库主要探讨 **BERT+ Attention 机制**在文本相似度，自然语言推理以及阅读理解等任务上的表现，并进行分析。 本仓库使用的数据集包括：

- Quora Question Pairs -- 英文
- 蚂蚁金服文本相似度数据集  -- 中文
- SQuAD -- 英文
- 百度阅读理解 -- 中文 ; <http://lic2019.ccf.org.cn/read>
- SNLI
- MultiNLI
